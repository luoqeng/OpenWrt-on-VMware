# OpenWrt-on-VMware
OpenWrt 本身获得自动翻墙能力，所有连入该路由的设备都可无障碍访问被墙的站点。

虚拟化软件 `vmwrae esxi hyper-v virtualbox kvm`，创建虚拟机时添加两张网卡，WAN 接口 NAT 或者 桥接模式，LAN 接口桥接。
